# My_Recipe
记录一些简单的家常菜谱

## 三汁焖锅
### 食材
素食：洋葱，青椒，红椒，芹菜，土豆，藕片，红薯，蒜瓣，等
肉类：鸡翅，虾，鹌鹑蛋，五花肉，等
酱汁：黄豆酱，蕃茄酱，蚝油，韩式辣椒酱，

### 做法
1. 素食切块，加适量油，辣椒粉，盐拌匀腌制十分钟
2. 肉类按照自己喜好腌制
3. 锅内刷黄油将素菜铺于下层，上层码肉类
4. 小火焖制10min
5. 黄豆酱2、蕃茄酱2、蚝油2、韩式辣酱1、生抽1、蜂蜜1、淀粉1调制酱汁
6. 酱汁铺于最上层，继续焖煮5min
7. 搅拌均匀
